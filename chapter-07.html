メモリーカード消滅事件
── ソニー、SD／CFexpress全停止

2026年3月27日、ソニーが日本国内のSDカードおよびCFexpressメモリーカードの受注を一斉停止した ── 消費者に直接波及した最も衝撃的な出来事のひとつ。

ソニーマーケティングの案内（2026/3/27）

半導体（メモリ）などの世界的不足により、CFexpressメモリーカードおよびSDメモリーカードは当面供給がご要望に追いつかない見通し。当社正規取扱店およびソニーストアでの受注を同日以降一時停止する。

― ソニーマーケティング

対象製品（ほぼ全品）

分類容量
CFexpress Type A240 / 480 / 960 / 1920GB
CFexpress Type B240 / 480GB
TOUGHシリーズ SD64 / 128 / 256GB
中間グレード SD128 / 256 / 512GB
V30規格 SD64 / 128GB
⚠ 残存出荷されるのは2製品だけ
960GB Type Bカードと一部の入門SF-UZシリーズSDのみ。実質的には「ほぼ全製品」が受注停止。

ソニーは「再開時期は供給状況を見ながら判断する」と述べるにとどまり、具体的な日取りを示していない。

Petapixelの見出し

SSD不足がついに写真業界にも最初の犠牲者をもたらした。ソニーが最初であるのは確かだが、最後ではないだろう。

― Petapixel

カメラ・動画業界への波紋

CFexpress Type A/Bは、プロ向けのα1・FX3・FX6・Nikon Z9・Canon R1/R5シリーズなどハイエンドカメラの高速記録用媒体として必須。ProRes録画用の高速SDXC、V30規格の動画用SDも、ミラーレス・ハンディカムで日常的に使われる。現場撮影中に「追加のメディアが買えない」という事態が現実に起き得る時代になったのだ。

⚠ 業界の連鎖懸念
同様の動きはサンディスクにも及び、Kioxia系のWestern Digital／SanDisk連合、Lexar、Samsungなどもメモリーカードの出荷調整や値上げに踏み切るかどうかが業界で注視されている。

プロはどう動くか

  • 在庫確保：次回案件用に多めに確保しておく。ただし買い占め批判は避ける
  • 機材ダブル化：デュアルカードスロット機を使い、メモリ交換頻度を落とす
  • エッジSSD運用：現場でMagSafe/USB-C直結の外付けSSDへオフロード
  • レンタル活用：短期案件ではカードもレンタルで済ませる
▶ この章の要点
  • 2026/3/27、ソニーがSD/CFexpressの国内受注を一斉停止
  • 対象は4容量のType A・2容量のType B・TOUGH/中間/V30 SD ── ほぼ全製品
  • 再開時期は未定
  • サンディスク他も出荷調整の可能性、業界全体が注視
